# 📺 M3U Editör Pro

> IPTV M3U playlist dosyalarını kolayca yönetin, düzenleyin ve filtreleyin

[English](README.md) | **Türkçe**

## ⚡ Hızlı Başlangıç

### Windows
```cmd
git clone https://github.com/kullaniciadi/m3uedit.git
cd m3uedit
run.bat
```

### Linux/Mac
```bash
git clone https://github.com/kullaniciadi/m3uedit.git
cd m3uedit
chmod +x run.sh
./run.sh
```

Tarayıcınızda otomatik olarak açılacak: http://localhost:8501

## ✨ Özellikler

- 🌐 URL'den veya dosyadan M3U yükleme
- 🇹🇷 Akıllı Türk kanal filtresi
- ✏️ İnteraktif tablo düzenleme
- 💾 M3U, JSON, CSV export
